# Безопасная газовая плита с компьютерным зрением

## Что это?

Система, которая следит за газовой плитой и предупреждает об опасных ситуациях.

## Как работает

### 1. Контроль газа
Камера смотрит на счетчик газа и определяет:
- **Есть ли изменения** на счетчике (горит газ или нет)
- Если газ включен, засекает время

### 2. Контроль человека
Вторая камера отслеживает кухню:
- Был ли человек в кадре за последние 10 минут
- Распознавание присутствия через компьютерное зрение

### 3. Оповещение
Если газ включен **И** человека нет на кухне больше 10 минут → 
📨 **Телеграм-бот отправляет предупреждение**

## Схема работы

```
Камера счетчика → Видит изменения → Газ включен
                                    ↓
Камера кухни → Нет человека 10 мин → ОПАСНО!
                                    ↓
                            Телеграм сообщение
```

## 🎯 Для чего нужно

- **Безопасность** — предотвращение утечек газа
- **Спокойствие** — можно не переживать, если забыли выключить газ
- **Автоматизация** — система работает без участия человека

## Разработка модели распознавания

### Проблема
Чтобы научить модель видеть изменения на счетчике, нужны размеченные реальные фото. 
Размечать вручную — долго и дорого.

### Решение

#### Создали синтетический датасет
- Сгенерировали 50 000+ искусственных изображений цифр
- С разными шрифтами, поворотами, освещением
- Уже размеченные

#### Обучили модель на синтетике
Модель научилась распознавать цифры на счетчике.

#### Автоматическая разметка реальных фото
- Собрали реальные фото со счетчиков
- Модель сама предсказала цифры
- Проверили только сомнительные случаи

#### Валидационный датасет готов
**Экономия времени на разметку: до 90%**

### Результат

```
Синтетика → Обучение модели → Авторазметка реальных фото
                                    ↓
                            Проверка сложных случаев
                                    ↓
                          Валидационный датасет ✓
```
