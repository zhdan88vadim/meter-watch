## How to Import This Dashboard

### Option 1: Import via JSON (Recommended)

1. **Access Grafana:**
   - Open http://localhost:3000
   - Login with `admin` / `admin`

2. **Import Dashboard:**
   - Click the **+** icon in the left sidebar
   - Click **Import**
   - Paste the JSON above into the text box
   - Click **Load**

3. **Configure Data Source:**
   - Select **PostgreSQL** from the dropdown
   - Click **Import**

### Option 2: Save as File and Import

1. **Save the JSON to a file:**
```bash
mkdir -p grafana/dashboards
cat > grafana/dashboards/meter_watch_dashboard.json << 'EOF'
[paste the JSON here]
EOF
```

2. **Import via API** (advanced):
```bash
# Get your API key from Grafana first
curl -X POST http://localhost:3000/api/dashboards/db \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d @grafana/dashboards/meter_watch_dashboard.json
```

## Troubleshooting

If panels show "No data", check:

1. **Data exists:**
```sql
SELECT COUNT(*) FROM meter_readings;
SELECT COUNT(*) FROM activity_logs;
```

2. **Table names are correct:** Your tables are `activity_logs` and `meter_readings`

3. **Time range:** Adjust the time picker to include your data

4. **Data source connection:** Go to Data Sources → PostgreSQL → Save & Test
