Method 1: Export from Grafana UI (Easiest)
Open your dashboard

Click the Settings (gear icon) ⚙️ in the top right

Select JSON Model from the left sidebar

Click Copy to Clipboard or Download JSON

Save the file with .json extension

Alternative quick way:

Open your dashboard

Click the Share icon (arrow pointing up) at the top

Go to the Export tab

Click Save to file or Copy to clipboard
